# 角色设定提示词模板

> Character Design Prompt Template
> 基于图片自动生成人物详细设定
> 适用于游戏、动漫、小说角色创作

---

## 一、模板概述

本模板旨在帮助用户根据上传的角色图片，自动生成完整、专业的角色设定文档。模板设计参考了多个优秀案例，涵盖角色命名、外貌特征、职业定位、武器类型、攻击特性、核心技能、性格风格及视觉基准等多个维度。

通过本模板，您可以快速为任何角色图片生成符合专业标准的设定文档，适用于游戏开发、动漫创作、小说写作等多种场景。

模板的核心优势在于其结构化和系统性。每个模块都有明确的填写要求和示例参考，确保生成的角色设定既具有丰富的细节，又能保持风格统一。无论是古风侠客、科幻未来、魔幻奇幻还是现代都市题材，本模板都能提供恰当的描述框架和灵感启发。

## 三、各模块详细说明

### 3.1 角色名称

角色名称是整个设定的灵魂，需要根据图片中角色的整体气质和视觉印象来创作。名称应当与角色的外貌、技能特点或职业背景相呼应，形成统一的角色印象。中文名应富有意境，英文名或拼音则为国际化传播提供便利。

**命名原则：**
- 与角色视觉特征相呼应（如红色系统可命名为"赤"字开头）
- 与技能属性相关联（如水系技能可命名为"泽""海"等）
- 体现职业特点（如剑客可命名为"剑"字相关）
- 富有文化底蕴或诗意雅韵

### 3.2 外貌特征

外貌特征是角色设定中最为直观的部分，需要细致观察图片中的每一个细节。描述应当从整体到局部，从上到下，层层递进，形成清晰的视觉印象。

**描述要点：**
- **面部特征**：眼神、眉毛、表情、面部饰物等
- **发型与头饰**：发色、发型、发饰、束发方式等
- **服饰细节**：颜色搭配、材质质感、特殊设计、装饰图案等
- **身体特征**：体型、姿态、肌肉线条等
- **特殊元素**：伤痕、纹身、光环、能量流动等

### 3.3 职业/攻击特性

职业定位决定了角色在战斗中的角色和战术风格。这一部分需要根据角色的服饰、武器和整体气质进行推断，并结合具体的世界观设定。

**常见职业类型：**
| 职业类型 | 特点 |
|----------|------|
| 剑修/剑客 | 敏捷型输出，注重速度和精准度 |
| 体术师/武僧 | 近身格斗，注重力量和防御 |
| 法师/术士 | 远程输出，注重范围伤害和控制 |
| 刺客/影者 | 爆发输出，注重隐匿和致命一击 |
| 特殊职业 | 根据世界观自定义，如"御灵体术师""时空缉捕司执事"等 |

### 3.4 核心技能

核心技能是角色设定的精华所在，需要为角色设计 2-3 个具有独特效果和视觉表现的技能。每个技能都应当有响亮的名称和详细的效果描述。

**技能设计原则：**
1. 技能名称要有识别度，体现技能的核心特点
2. 效果描述要具体，包含视觉表现和实际作用
3. 技能之间要有区别度，分别对应不同的战斗场景
4. 考虑技能的连招和组合可能性

### 3.5 武器类型

武器是角色战斗能力的重要载体，需要根据图片中角色所持或配备的武器进行详细描述。

**武器类型示例：**
| 类型 | 说明 |
|------|------|
| 冷兵器 | 剑、刀、枪、斧等传统武器 |
| 法器/道具 | 符咒、法杖、印章、套符等 |
| 活物武器 | 灵兽、式神、特殊生物等（如"太始灵鲤"） |
| 科技武器 | 粒子切割器、能量武器、机械装备等 |
| 特殊装备 | 护腕、护甲、饰品具有战斗功能等 |

### 3.6 性格/风格（可选）

性格特点为角色注入灵魂，使其从单纯的视觉形象升华为有血有肉的人物。

**描述方向：**
- 基本性格特征：冷峻、热血、温柔等
- 价值观与信念：重情义、追求正义、守护弱者等
- 行为风格：优雅、沉稳、活泼等
- 特殊状态下的表现：战斗中的变化、情感爆发等

### 3.7 视觉基准锁定（可选）

视觉基准锁定为角色的视觉呈现提供统一的指导方向。

**视觉要素：**
- 整体画风：写实、卡通、水墨、赛博朋克等
- 色彩基调：主色、辅助色、强调色等
- 光影效果：霓虹光、自然光、气氛光等
- 特殊效果：粒子、运动模糊、流光等
- 材质表现：金属光泽、布料质感、皮肤纹理等

## 四、案例参考

### 案例一：赤鲤（古风侠客）

**角色名称：** 赤鲤（Chi Li）

**外貌特征：** 少年英侠，剑眉深邃，发丝因疾走而张扬。身穿红白相间的无袖修身战袂，双臂套有黑金玄铁护腕，那是为了驾驭狂暴灵力而制的锁具。

**职业/攻击特性：** "御灵体术师"。他并不使用常规冷兵器，而是以怀中的"太始灵鲤"作为能量核心，通过肉搏结合灵波进行大范围打击。

**核心技能：**
- 【赤波冲坠】：借锦鲤之势发动超高速俯冲，落地瞬间爆发出圆环状的赤红冲击波。
- 【锦鳞万叠】：周身红色丝带化为坚硬如钢的鳞片护甲，反弹一切冲击。
- 【龙门跃·断空】：极高跳跃后的重拳挥击，伴随巨大的灵鲤虚影撕裂空间。

**武器类型：** 活物武器（太始灵鲤）+ 强化护腕。锦鲤能改变重力，护腕能压缩灵能。

---

### 案例二：云澈（悲情剑客）

**角色名称：** 云澈

**外貌特征：** 少年剑客，剑眉星目却带着破碎感，泪水盈眶。束发高耸，系红色抹额，身着深蓝与银灰色相间的精锐轻甲，饰有繁复的云纹浮雕。

**职业/攻击特性：** 敏捷型剑修。擅长"瞬步"与"御剑术"，攻击节奏极快，追求一击必杀的精准度。

**核心技能：**
- 【苍龙破浪】：幽蓝剑气爆发，形成巨大的剑气浪潮。
- 【瞬影连斩】：多重残影切割，在短时间内造成多次连续攻击。

**武器类型：** 龙纹古剑（长剑，刃薄而利，具物理切割与能量传导双重属性）。

**性格/风格：** 坚韧、重情义、在极度悲愤中爆发极强战力。

---

### 案例三：雷泽（现代科幻）

**角色名称：** 雷泽

**职业：** 时空缉捕司·见习执事 / 孤傲的都市散修

**外貌特征：** 银灰色凌乱高马尾，头戴红宝石金纹发冠，双鬓垂下流光丝线。眼神深邃，眼角带有泪滴状的感光传感器。身着白灰相间的科技质感长衫。

**武器类型：** "空鸣"粒子震动唐刀（平时伪装成咖啡杯上的线条纹路，战斗时瞬间凝聚）。

**攻击特性：** 极速、精准、切割。利用高频震动粒子斩断一切物质，打击感表现为整齐的切割面与湛蓝色电弧。

**核心技能：**
- 【瞬影·绝切】：在 0.1 秒内跨越 10 米距离进行横向斩击。
- 【流光结界】：发丝间的能量线散开，形成半径 3 米的防御/感知网。

**性格表现：** 外表冷淡、实则重情，即使在最激烈的战斗中也要保持"优雅的日常感"（如护住手中的咖啡）。

**视觉基准锁定：** 超写实电影级 CG 风格。高饱和度的霓虹光效（青蓝、橘红），伴随大量的流光粒子和运动模糊，景深极浅，强调面部纹理与液态金属般的粒子质感。

## 五、使用指南

### 操作步骤

1. 上传角色图片，确保图片清晰、角色形象完整
2. 使用本模板的提示词框架，要求 AI 进行分析
3. 根据图片特点选择适当的世界观和风格
4. 检查生成的设定，确保各模块之间逻辑一致
5. 根据需要进行调整和补充

### 注意事项

- 技能名称应具有识别度，避免过于通用
- 外貌描述要与图片严格对应，不要凭空想象
- 武器和技能要与角色整体风格协调一致
- 可选模块根据实际需要决定是否填写
