# 视频生成平台参数与限制

> 整理 Grok Imagine、Nano Banana 2、Seedance 2.0 等平台规格。

---

## Grok Imagine (Kie AI)

> 基于 Kie AI OpenAPI 文档整理，版本 8.1.0

### 平台概述

| 维度 | 说明 |
|------|------|
| **平台** | Kie AI (https://kie.ai) |
| **认证方式** | API Key (Bearer Token) |
| **API 基础URL** | https://api.kie.ai |
| **任务创建端点** | POST /api/v1/jobs/createTask |
| **任务查询端点** | GET /api/v1/jobs/recordInfo?taskId={taskId} |

### 生成模式

| 模式 | Model 名称 | 功能 |
|------|-----------|------|
| 文生视频 (T2V) | `grok-imagine/text-to-video` | 纯文本生成视频 |
| 图生视频 (I2V) | `grok-imagine/image-to-video` | 图片转视频 |
| 视频扩展 | `grok-imagine/extend` | 延长已有视频 |

### 通用参数

| 参数 | 类型 | 可选值 | 默认值 | 说明 |
|------|------|--------|--------|------|
| `duration` | string | `"6"`, `"10"` | `"10"` | 视频时长（秒） |
| `resolution` | string | `"480p"`, `"720p"` | `"720p"` | 视频分辨率 |
| `aspect_ratio` | string | `"2:3"`, `"3:2"`, `"1:1"`, `"16:9"`, `"9:16"` | `"9:16"` | 画面比例 |
| `mode` | string | `"fun"`, `"normal"`, `"spicy"` | `"normal"` | 生成风格模式 |
| `prompt` | string | 最大 5000 字符 | - | 视频描述提示词（支持英文） |
| `callBackUrl` | string (URI) | - | - | 任务完成回调 URL |

### 风格模式说明

| 模式 | 说明 |
|------|------|
| `fun` | 更有创意和趣味的解读 |
| `normal` | 平衡方法，具有良好的运动质量 |
| `spicy` | 更有活力和强烈的运动效果（外部图片不可用） |

### 图生视频 (I2V) 特殊参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `image_urls` | array[string] | 外部图片 URL 列表（最多 7 张），不能与 task_id 同时使用 |
| `task_id` | string | 之前生成的 Grok 图像任务 ID，不能与 image_urls 同时使用 |
| `index` | integer | 使用 task_id 时指定图像索引 (0-5)，默认 0 |

### I2V 图片规格

| 维度 | 规格 |
|------|------|
| **支持格式** | JPEG、PNG、WEBP |
| **单张大小** | ≤ 10MB |
| **数量上限** | 7 张 |
| **spicy 模式** | 外部图片不支持，自动降为 normal |

### I2V 多图模式

- **单图模式**：视频宽高参考图片宽高，aspect_ratio 参数无效
- **多图模式**：可自定义 aspect_ratio 指定画面比例

### 视频扩展参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `task_id` | string | ✅ | - | 之前成功的视频生成任务 ID |
| `prompt` | string | ✅ | - | 描述视频如何扩展的提示词（支持中英文） |
| `extend_at` | number | ✅ | 0 | 扩展起点位置 |
| `extend_times` | string | ✅ | "10" | 扩展时长 "6" 或 "10" 秒 |

### 视频扩展限制

- 只能扩展由 Kie AI 生成的视频，不支持外部视频
- 原始视频生成必须成功完成
- task_id 必须来自 `grok-imagine/text-to-video` 或 `grok-imagine/image-to-video`
- 扩展时长越长，生成所需时间相应增加

### 视频扩展示例

```python
# 扩展视频 10 秒
result = await client.extend_video(
    task_id="task_grok_xxx",
    prompt="镜头缓慢推进，展示主角走进森林深处",
    extend_at=0,
    extend_times="10"
)
```

### API 响应状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 - 请求已成功处理 |
| 401 | 未授权 - 缺少身份验证凭据或凭据无效 |
| 402 | 额度不足 - 账户额度不足，无法执行该操作 |
| 404 | 未找到 - 请求的资源或接口不存在 |
| 422 | 校验错误 - 请求参数未通过校验检查 |
| 429 | 请求受限 - 已超过该资源的请求频率限制 |
| 455 | 服务不可用 - 系统目前正在维护中 |
| 500 | 服务器错误 - 处理请求时发生了意外错误 |
| 501 | 生成失败 - 内容生成任务失败 |
| 505 | 功能禁用 - 请求的功能目前已禁用 |

### 任务状态

| 状态 | 说明 |
|------|------|
| `waiting` | 已排队 |
| `queuing` | 队列中 |
| `generating` | 生成中 |
| `success` | 生成成功 |
| `fail` | 生成失败 |

### 平台限制

- 提示词最大 5000 字符
- 图生视频最多 7 张图片
- 外部图片不支持 spicy 模式
- 支持回调 URL 异步通知
- 建议生产环境使用 callBackUrl 而非轮询

## Nano Banana 2 (Kie AI)

> 基于 Kie AI OpenAPI 文档整理，API 端点：`https://api.kie.ai/api/v1/jobs/createTask`

### 平台概述

| 维度 | 说明 |
|------|------|
| **平台** | Kie AI (https://kie.ai) |
| **认证方式** | API Key (Bearer Token) |
| **API 基础URL** | https://api.kie.ai |
| **任务创建端点** | POST /api/v1/jobs/createTask |
| **任务查询端点** | GET /api/v1/jobs/recordInfo?taskId={taskId} |
| **Model 名称** | `nano-banana-2` |

### 生成模式

| 模式 | 功能 | 说明 |
|------|------|------|
| 文生图 (T2I) | 纯文本生成图像 | 最大 20000 字符提示词 |
| 图生图 (I2I) | 图片转换风格 | 最多 **14 张**参考图 |

### 通用参数

| 参数 | 类型 | 可选值 | 默认值 | 说明 |
|------|------|--------|--------|------|
| `prompt` | string | 最大 20000 字符 | - | 图像描述提示词（必填） |
| `aspect_ratio` | string | 见下表 | `auto` | 画幅比例 |
| `resolution` | string | `1K`, `2K`, `4K` | `1K` | 输出分辨率 |
| `output_format` | string | `jpg`, `png` | `jpg` | 输出格式 |
| `image_input` | array[string] | 最多 14 张图片 URL | - | 参考图片列表 |
| `callBackUrl` | string (URI) | - | - | 任务完成回调 URL |

### 画幅比例选项

| 比例 | 说明 | 适用场景 |
|------|------|----------|
| `auto` | 自动 | 根据提示词自动选择 |
| `1:1` | 正方形 | 头像、产品图 |
| `16:9` | 横版 | 视频、分镜、场景 |
| `9:16` | 竖版 | 手机壁纸、短视频 |
| `4:3` | 经典横版 | 照片 |
| `3:4` | 经典竖版 | 人像 |
| `21:9` | 电影宽屏 | 全景、鸟瞰 |
| `2:3` | 竖版 | 海报 |
| `3:2` | 横版 | 照片 |
| `1:4` | 超长竖版 | 条漫 |
| `4:1` | 超长横版 | 条漫 |
| `1:8` | 极长竖版 | 长图 |
| `8:1` | 极长横版 | 长图 |
| `4:5` | 社交竖版 | Instagram |
| `5:4` | 社交横版 | Instagram |

### 图片输入规格

| 维度 | 规格 |
|------|------|
| **支持格式** | JPEG、PNG、WEBP |
| **单张大小** | ≤ 30MB |
| **数量上限** | **14 张** |
| **输入方式** | URL（上传后获取） |

### 特殊生成能力

| 功能 | 说明 | 命令 |
|------|------|------|
| 角色三视图 | 上半身/正面全身/背面全身并排 | `tri-view` |
| 白底图 | 提取主体生成白底图 | `white-bg` |
| 场景鸟瞰图 | 超远距离鸟瞰视角 | `bird-view` |
| 场景全景图 | 210度全景视图 | `panoramic` |
| 九宫格分镜 | 3x3 完整故事板 | `9grid` |
| 三宫格分镜 | 3格分镜故事板 | `3grid` |
| 图像放大 | 超高清放大至 4K | `upscale` |
| 批量生成 | 批量处理多张图 | `batch` |

### API 响应状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 401 | 未授权 |
| 402 | 额度不足 |
| 404 | 未找到 |
| 422 | 校验错误 |
| 429 | 请求受限 |
| 455 | 服务不可用 |
| 500 | 服务器错误 |
| 501 | 生成失败 |
| 505 | 功能禁用 |

### 使用建议

- 九宫格生成耗时较长，建议设置 `timeout=900`
- 生产环境建议使用 `callBackUrl` 而非轮询
- 参考图片建议先上传获取 URL 再传入
