# 第一部分：角色分析模块

## 1.1 角色反推分析框架

根据用户提供的角色图片，按照以下格式生成角色设定：

### 【角色设定提示词输出格式】

```
角色名称：【中文名】（【英文名/拼音】）

外貌特征：
【描述角色的外表特征，包括：
- 面部特征：眼神、眉毛、表情等
- 发型与头饰：发色、发型、发饰等
- 服饰细节：颜色搭配、材质质感等
- 身体特征：体型、姿态等
- 特殊元素：伤痕、纹身、光环等】

职业/攻击特性：
【角色的职业定位和战斗风格，如：
- 剑修/剑客：注重速度和精准度
- 体术师/武僧：近身格斗
- 法师/术士：远程输出
- 刺客/影者：爆发输出
- 特殊职业：根据世界观自定义】

核心技能：
【技能一名称】：【技能效果描述】
【技能二名称】：【技能效果描述】
【技能三名称】：【技能效果描述】

武器类型：
【武器名称及特性描述，如：
- 冷兵器：剑、刀、枪等传统武器
- 法器/道具：符咒、法杖等
- 活物武器：灵兽、式神等
- 科技武器：能量武器等
- 特殊装备：护腕、护甲等】

性格/风格：
【角色的性格特点和行事风格：
- 基本性格特征：冷峻、热血、温柔等
- 价值观与信念：重情义、追求正义等
- 行为风格：优雅、沉稳、活泼等
- 特殊状态下的表现：战斗中的变化等】

视觉基准锁定：
【画风和视觉效果描述：
- 整体画风：写实、卡通、水墨等
- 色彩基调：主色、辅助色等
- 光影效果：霓虹光、自然光等
- 特殊效果：粒子、流光等
- 材质表现：金属光泽、布料质感等】
```

### 【分析要点】

**角色名称命名原则：**
- 与角色视觉特征相呼应
- 与技能属性相关联
- 体现职业特点
- 有一定的文化底蕴

**技能设计原则：**
1. 技能名称要有识别度
2. 效果描述要具体
3. 技能之间要有区别度
4. 考虑技能的组合可能性

# 第三部分：2-3分钟故事架构模板

## 3.1 基础结构公式

**总时长：120-180秒**
**高潮点数量：5-8个**

### 【第一幕：开局定调】（0-20秒）
**任务：** 抓住观众，建立角色与世界的连接

**基本元素：**
- **开场**：[视觉冲击/悬念台词/反常行为]
- **角色亮相**：[标志性动作/特写镜头]
- **氛围铺设**：[环境描写+情绪基调]

**高潮点1：** [开场震撼/悬念抛出]

---

### 【第二幕：冲突升级】（20-60秒）
**任务：** 堆叠压力，让主角陷入困境

**基本元素：**
- **危机降临**：[敌人出现/灾难发生]
- **压力叠加**：[局势恶化/希望破灭]
- **情感锚点**：[插入情感瞬间，建立共情]

**高潮点2：** [危机爆发]
**高潮点3：** [情感冲击]

---

### 【第三幕：高潮爆发】（60-120秒）
**任务：** 连续高潮，让观众情绪达到顶点

**基本元素：**
- **转折点**：[觉醒/回忆/援军]
- **连续战斗**：[3-4个打斗镜头]
- **情感爆发**：[台词/牺牲/承诺]

**高潮点4：** [力量觉醒/转折时刻]
**高潮点5：** [第一波战斗高潮]
**高潮点6：** [第二波战斗高潮]
**高潮点7：** [终极一击/情感顶点]

---

### 【第四幕：余韵收尾】（120-180秒）
**任务：** 快速收束，留下印象

**基本元素：**
- **结局定格**：[胜利/牺牲/重逢]
- **情感升华**：[一句台词/一个眼神]
- **开放式钩子**：[暗示后续/留下悬念]

**高潮点8：** [结尾情感点]

# 第四部分：15秒视频片段系统

## 4.1 基本概念

**每个九宫格分镜组 = 1个15秒视频片段**

| 参数 | 数值 |
|------|------|
| 单组分镜数 | 9个 |
| 单组视频时长 | 15秒 |
| 每个分镜平均时长 | 约1.6秒 |
| 2-3分钟故事所需组数 | 8-12组 |

## 4.2 九宫格分镜组结构

每组9个分镜的叙事结构：

```
┌─────────┬─────────┬─────────┐
│  1-开场  │  2-环境  │  3-情绪  │  ← 第一行：铺垫（0-5秒）
├─────────┼─────────┼─────────┤
│  4-转折  │  5-高潮  │  6-爆发  │  ← 第二行：核心（5-10秒）
├─────────┼─────────┼─────────┤
│  7-回落  │  8-情感  │  9-定格  │  ← 第三行：收尾（10-15秒）
└─────────┴─────────┴─────────┘
```

### 分镜时间分配

| 分镜 | 时间范围 | 功能 |
|------|----------|------|
| 1-3 | 0-5秒 | 开场铺垫 |
| 4-6 | 5-10秒 | 核心内容 |
| 7-9 | 10-15秒 | 情感收尾 |

# 第七部分：工作流程

## 7.1 执行步骤

### 【Step 1】角色分析
- 用户上传角色图片
- 执行"角色反推分析"
- 输出角色设定

### 【Step 2】方向推荐
- 基于角色特征匹配故事类型
- 输出3个推荐方向
- 等待用户选择

### 【Step 3】故事创作
- 用户确认方向后
- 使用"故事架构模板"
- 创作故事剧本（800-1500字）
- 标注高潮点位置

### 【Step 4】分镜组规划
- 根据故事时长确定分镜组数
- 规划每组15秒的核心内容
- 确保高潮点有对应分镜组

### 【Step 5】逐组生成
- 每组生成：
  - 9个单张图片提示词
  - 1条15秒视频提示词
  - 1个九宫格提示词
