# 成片装配工作流

## 概述

成片装配将分镜视频拼接为完整的短剧成品。

## 功能特性

- **智能拼接** - 自动按顺序拼接视频片段
- **转场效果** - 支持多种转场效果
- **音视频同步** - 音画同步处理
- **多格式输出** - 支持多种分辨率和格式

## 装配流程

### Step 1: 片段审查

逐个审查每个分镜视频，选择最佳版本。

**决策点**:
- 画面质量
- 动作流畅度
- 与前后片段的连贯性

### Step 2: 顺序确认

确认视频片段的播放顺序。

**检查项**:
- 时间线逻辑
- 场景转换流畅度
- 叙事连贯性

### Step 3: 视频拼接

使用 ffmpeg 进行高质量拼接。

**编码选项**:
- `copy` - 直接复制，不重新编码（速度快）
- `h264` - H.264 重新编码（质量高）

### Step 4: 音视频合成

如有独立音频，进行音视频合成。

**处理项**:
- 环境音
- 人声配音
- 音效

### Step 5: 最终输出

生成最终成片。

**输出规格**:
- 分辨率: 1080x1920 (9:16) / 1920x1080 (16:9)
- 格式: MP4
- 编码: H.264

## 使用方式

### 自然语言调用

- "拼接所有分镜视频"
- "导出最终短剧"
- "生成成片"

### 命令行调用

```bash
python3 scripts/workflow.py assemble \
  --clips video1.mp4 video2.mp4 video3.mp4 \
  --output final_drama.mp4
```

### 高级用法

```bash
# 带转场效果
python3 scripts/video_assembler.py concat \
  --inputs clip1.mp4 clip2.mp4 clip3.mp4 \
  --output final.mp4

# 添加音频
python3 scripts/video_assembler.py add-audio \
  --video final.mp4 \
  --audio background_music.mp3 \
  --output final_with_audio.mp4 \
  --volume 0.3
```

## 输出规格

| 参数 | 数值 |
|------|------|
| 格式 | MP4 |
| 编码 | H.264 |
| 音频 | AAC 128kbps |
| 分辨率 | 1080x1920 (竖屏) / 1920x1080 (横屏) |
| 帧率 | 30fps |

## 质量检查

### 画面检查
- ✅ 拼接处无明显跳跃
- ✅ 色彩一致
- ✅ 无黑边/变形

### 音频检查
- ✅ 音画同步
- ✅ 无爆音/杂音
- ✅ 音量平衡

### 整体检查
- ✅ 叙事流畅
- ✅ 时长合理
- ✅ 结尾完整
