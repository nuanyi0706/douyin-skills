# 资产生成工作流

## 概述

资产生成为短漫剧生成角色、场景、道具等视觉资产。

## 资产类型

### 角色三视图

生成角色的三个视角图，用于保持角色一致性。

**输出结构**:
- 左边：人物上半身
- 中间：人物正面全身
- 右边：人物背面全身

**适用场景**:
- 新角色首次登场
- 角色服装/造型变更后
- 后续分镜的参考基准

**分辨率**: 建议 2K

### 角色无背景全身图

提取角色主体，生成无背景的全身图。

**适用场景**:
- 与场景合成时便于抠图
- 制作角色立牌
- 换场景/换服装时参考

### 场景鸟瞰图

超远距离俯视角度的场景图。

**特点**:
- 无缝拼接，无畸变
- 超高清 4K 分辨率
- 照片级真实感

**适用场景**:
- 建立镜头
- 交代地理位置
- 全景展示

### 场景全景图

210度超广角环绕视图。

**特点**:
- 等距柱状投影
- 完整环绕视角
- 超宽画幅 21:9

**适用场景**:
- 环境展示
- 空间关系交代
- 沉浸式氛围营造

### 道具白底图

提取道具主体，生成纯白背景图。

**适用场景**:
- 道具特写
- 物品细节展示
- 便于后期合成

## 生成流程

1. **准备描述** → 提供资产详细描述
2. **选择类型** → 确定资产生成类型
3. **调用 API** → Nano Banana 2 生成
4. **结果输出** → 返回图像 URL

## 使用方式

### 自然语言调用

- "生成角色三视图"
- "生成场景鸟瞰图"
- "生成道具白底图"

### 命令行调用

```bash
# 角色三视图
python3 scripts/workflow.py generate-asset \
  --type character-tri-view \
  --description "古风少女，白衣汉服" \
  --resolution 2K

# 场景鸟瞰图
python3 scripts/workflow.py generate-asset \
  --type scene-bird-view \
  --image scene.png \
  --resolution 4K

# 道具白底图
python3 scripts/workflow.py generate-asset \
  --type white-bg \
  --image prop.png \
  --description "玉佩"
```

## 质量规范

- ✅ 角色一致性：三视图风格完全统一
- ✅ 无背景：主体清晰，边缘干净
- ✅ 高分辨率：最低 2K，建议 4K
- ❌ 无水印：无时间码、无字幕
- ❌ 无风格漂移：全片风格保持一致
