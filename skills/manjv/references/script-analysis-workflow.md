# 剧本分析工作流

## 概述

剧本分析是 manjv 的第一步，深度分析小说文本，提取创作所需的核心要素。

## 功能特性

- **角色提取** - 自动识别并提取文本中的角色信息
- **场景提取** - 识别并整理场景列表
- **道具提取** - 提取关键道具和物品
- **故事结构分析** - 分析幕、场次、高潮点

## 提取内容

### 角色信息

```json
{
  "name": "角色名称",
  "description": "外貌特征描述",
  "personality": "性格特点",
  "role": "在故事中的定位"
}
```

### 场景信息

```json
{
  "name": "场景名称",
  "location": "具体地点",
  "time": "时间段",
  "mood": "氛围基调"
}
```

### 道具信息

```json
{
  "name": "道具名称",
  "description": "外观描述",
  "significance": "剧情意义"
}
```

## 使用方式

### 自然语言调用

- "分析这个小说片段"
- "提取剧本中的角色"
- "列出所有场景"

### 命令行调用

```bash
python3 main.py analyze --script "小说文本..."
```

## 输出格式

分析完成后输出结构化的 JSON 报告，包含：
- 角色列表
- 场景列表  
- 道具列表
- 故事结构
- 创作建议
