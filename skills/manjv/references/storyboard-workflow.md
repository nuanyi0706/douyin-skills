# 分镜绘制工作流

## 概述

分镜绘制基于剧本生成分镜故事板，将文字剧本可视化为图像序列。

## 分镜类型

### 9宫格分镜

3x3 完整故事板，包含9个分镜。

**结构**:
```
┌─────────┬─────────┬─────────┐
│  1-开场 │  2-环境 │  3-情绪 │ ← 第一行：铺垫(0-5秒)
├─────────┼─────────┼─────────┤
│  4-转折 │  5-高潮 │  6-爆发 │ ← 第二行：核心(5-10秒)
├─────────┼─────────┼─────────┤
│  7-回落 │  8-情感 │  9-定格 │ ← 第三行：收尾(10-15秒)
└─────────┴─────────┴─────────┘
```

**适用场景**:
- 完整情节展示
- 15秒视频片段
- 复杂场景叙事

**分辨率**: 建议 4K

### 3宫格分镜

精简版3格故事板。

**结构**:
```
┌─────────┬─────────┬─────────┐
│  1-开场 │  2-发展 │  3-结尾 │
└─────────┴─────────┴─────────┘
```

**适用场景**:
- 快速预览
- 简单情节
- 短视频片段

**分辨率**: 建议 2K

## 分镜内容

每个分镜包含：
- **时间标签** - 如 1-1, 1-2, ...
- **镜头类型** - 特写/中景/全景
- **画面描述** - 人物动作、环境、光影
- **时长分配** - 每个分镜的秒数

## 生成流程

1. **剧本解析** → 提取分镜脚本
2. **风格设定** → 确定视觉基准
3. **提示词生成** → 构建专业提示词
4. **Nano Banana** → 生成9宫格图像
5. **结果输出** → 返回图像 URL

## 使用方式

### 自然语言调用

- "基于剧本生成9宫格分镜"
- "生成3宫格精简分镜"
- "生成分镜故事板"

### 命令行调用

```bash
# 9宫格分镜
python3 scripts/workflow.py generate-storyboard \
  --type 9grid \
  --content "完整的分镜描述..." \
  --style "写实电影风格" \
  --images scene.png character1.png character2.png \
  --resolution 4K

# 3宫格分镜
python3 scripts/workflow.py generate-storyboard \
  --type 3grid \
  --content "三格分镜内容..." \
  --resolution 2K
```

## 质量规范

- ✅ 风格统一 - 所有分镜色调、质感一致
- ✅ 构图专业 - 每个分镜构图优秀
- ✅ 标签清晰 - 角标编号清晰可读
- ❌ 无文字/字幕 - 纯视觉展示
- ❌ 无时间码/水印
- ❌ 无风格漂移
